Regulamentul CTF

Pentru a menține un mediu corect și funcțional pentru toți studenții, te rugăm să respecți următoarele reguli în timpul sesiunilor de antrenament.

1. Conduita pe Platformă

  • Fără atacuri asupra infrastructurii: Este interzisă orice formă de atac de tip DoS/DDoS, brute-force asupra formularului de login sau încercarea de a exploata platforma web (frontend/backend).
  • Ținta este mediul izolat: Acțiunile tale trebuie să se rezume strict la mediul Linux oferit prin conexiunea SSH la fiecare challenge.

2. Formatul Flag-urilor

  • Toate flag-urile respectă formatul: ATM_CTF{un_text_oarecare}.
  • Dacă găsești textul "felicitari_ai_rezolvat", flag-ul pe care trebuie să îl introduci în platformă este ATM_CTF{success}.
  • Flag-urile sunt case-sensitive (se face distincția între litere mari și mici).

3. Colaborare și Partajare

  • Este interzisă partajarea flag-urilor: Nu oferi flag-urile altor studenți. Scopul platformei este învățarea.
  • Colaborarea pentru a înțelege conceptele este încurajată, dar fiecare utilizator trebuie să descopere singur flag-ul pe mediul său.

4. Raportarea Erorilor

Dacă descoperi o eroare în platformă, o problemă la conexiunea SSH sau un challenge care nu funcționează conform descrierii, te rugăm să-mi raportezi problema imediat. Nu încerca să exploatezi acea eroare.
